# SNCF Max Confirmator
Un outil vous qui confirme automatiquement vos voyages SNCF Max.

# Comment m'ajouter ?
- Ouvrir une nouvelle fenêtre de navigation privée.
- Aller sur https://www.maxjeune-tgvinoui.sncf/sncf-connect et se connecter si besoin.
- Ouvrir les outils de développement. (Windows : `Ctrl + Shift + C`, Mac : `Cmd + Option + C`).
- Aller dans l'onglet "Application" des outils de développement.
- Aller dans Storage > Cookies > https://www.maxjeune-tgvinoui.sncf
- M'envoyer la valeur du cookie `auth`.
